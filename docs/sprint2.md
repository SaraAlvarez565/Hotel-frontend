# 🚀 Sprint 2 - Frontend del sistema

## Objetivo
Desarrollar la interfaz de usuario e integrar el backend con React.

## Funcionalidades implementadas

- Registro de usuarios
- Inicio de sesión
- Listado de productos
- Filtros por categoría
- Búsqueda de productos
- Detalle de producto
- Sistema de reservas
- Sistema de favoritos
- Panel administrativo
- Consumo de API REST

## Tecnologías utilizadas
- React
- Axios
- React Router DOM

## Resultado
El frontend quedó completamente funcional conectado al backend.
